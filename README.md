# SpaceFlow-AI
SpaceFlow AI 是一個智慧空間配置系統，通過自動化和優化算法（特別是強化學習）來有效配置和優化多邊形圖層中的空間布局。該系統能夠從DXF文件中讀取指定圖層的多邊形數據，根據配置文件的參數進行空間和走道的配置，並利用強化學習算法不斷優化空間布局，以滿足特定的約束條件和優化目標。


SpaceFlow AI 系統主要由以下幾個核心模組組成：

1. DXF 文件處理模組：負責匯入和解析DXF文件中的多邊形數據。
2. 配置文件處理模組：讀取並解析配置文件中的空間配置參數。
3. 空間與走道配置模組：根據解析出的數據生成初步的空間布局。
4. 約束條件管理模組：確保生成的空間布局符合所有設計約束。
5. 優化與強化學習模組：利用強化學習算法自動優化空間布局。
6. API 模組：提供對外的數據匯入、策略生成和評估介面。
7. 數據存儲與管理模組：管理系統中的各類數據，確保數據的可用性和一致性。
8. 結果輸出與可視化模組：將最終的空間配置結果轉換為用戶可理解的格式並進行可視化展示。
9. 用戶管理與安全模組：管理用戶訪問權限，確保系統安全。
